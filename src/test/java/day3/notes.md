## CSS Selectors

--------------------------------------------
There are types of Syntax used for CSS Selectors

    # the tag word is optional, but # or . is mandatory for all.
    # if there are multiple words after whitespace in a classname, 
        then we will add only the first word of the classname, 
        will leave the rest of the classname after whitespace.
        
    
    1. tag id =>    tag#id
    2. tag class => tag.classname 
    3. tag attribute => tag[attribute = 'value']
    4. tag class attribute ==> tag.classname[attribute = 'value']
